## 顺序表

定义

```c
#define InitSize 100	// 表长度的初始定义
typedef struct{
	ElemType *data;    // 动态分配数组的指针
    int MaxSize, length;	// 数组的最大容量和当前个数
} SqList;		// 定义一个顺序表

// ElemType 按照需要定义  int, char, float
```

C初始动态分配语句

```c
L.data = (ElemType*)malloc(sizeof(ElemType)*InitSize);
```
